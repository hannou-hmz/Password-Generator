# Password Generator

## Description
This is a secure and customizable password generator that allows users to generate strong passwords based on their preferences. Users can select different criteria, such as including uppercase letters, lowercase letters, numbers, and special characters. This project helps users generate secure passwords for various applications and accounts.

## Features
- **Customizable Options**: Users should select at least one of the following options:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
  if the users didnt select any of the options you will see a message "Choose one of the options..."
- **Password Length**: The user can specify the desired length of the password of min 9 characters.
- **Randomization**: All password components are randomly selected to ensure uniqueness and strength.
- **User-friendly Interface**: Simple and intuitive design to easily choose the desired options.

## Technologies Used
- **HTML**: For the structure of the web page.
- **CSS**: For styling and layout.
- **JavaScript**: For functionality and logic behind password generation.

## How to Use
1. **Copy this link to your browser** : " https://hannou-hmz.github.io/Password-Generator/"
2. **Choose Options**: Select at least one of the following options to customize the password:
   - Uppercase letters
   - Lowercase letters
   - Numbers
   - Special characters
3. **Set Password Length**: Use the input field to specify the desired password length (minimum of 9 characters recommended for security).
4. **Generate Password**: Click the “Generate” button to create a secure password.
5. **Copy Password**: Once the password is generated, you can copy it to your clipboard for use.
